# Ballerina by Example

[Ballerina][ballerina] is a general purpose, concurrent and strongly typed programming language with both textual and graphical syntaxes, optimized for integration.

Ballerina by Example is a hands-on introduction to Ballerina using annotated example programs. Check out the first example or browse the full list below.
